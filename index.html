Easy Installation Guide

Follow these 5 simple steps to install the extension in your Google Chrome browser.

01

Download the Package

Click the button above to download the extension ZIP file to your computer.

02

Extract the ZIP file

Right-click the downloaded .zip file and extract/unzip it into a folder on your drive.

03

Open Chrome Extensions

Open Google Chrome and navigate to the extensions page:

chrome://extensions/
04

Enable Developer Mode

In the top-right corner of the Extensions tab, toggle the "Developer mode" switch to ON.

05

Load Unpacked

Click "Load unpacked" in the top-left and select the extracted folder containing the extension files.
